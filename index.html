What is Material Design 3?

Material Design 3 (also called Material You) is Google's latest design system. It introduces dynamic color, updated typography, and an expressive, adaptive approach to UI components.

Why use a classless library?

Classless CSS libraries let you write clean, semantic HTML without adding utility classes to every element. Your markup stays readable and your styles stay in one place — the stylesheet.

Is it accessible?

Yes. Every interactive element uses focus-visible for keyboard accessibility, sufficient color contrast ratios, and semantic HTML structures throughout.

Tip: You can customize the entire library by overriding the CSS custom properties in :root. Change --md-primary to shift the entire color palette.
